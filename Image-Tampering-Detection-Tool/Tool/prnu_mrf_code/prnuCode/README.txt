 Contents
-------------------------------------------------------------------

(1) Forgery localization based on PRNU.
   In this package, the techniques described in [1] is implemented.
   See "demo_forgerylocalization.m" for more details.

(2) PRNU extraction. 
   The package contains a modified version of the code of DDE Lab, Binghamton University, NY [2]
   (http://dde.binghamton.edu/download/camera_fingerprint/)
   See "demo_estimateFingerprint.m" for more details.
     

[1] G. Chierchia, G. Poggi, C. Sansone and L. Verdoliva,
''A Bayesian-MRF Approach for PRNU-based Image Forgery Detection''
IEEE Transactions on Information Forensics and Security, 2014.

[2] M. Goljan, J. Fridrich, and T. Filler, 
Large Scale Test of Sensor Fingerprint Camera Identification.
Proc. of SPIE, Electronic Imaging, Media Forensics and Security XI, volume 7254, January 2009.
